# Countdown Timer

A simple and responsive countdown timer built using HTML, CSS, and JavaScript.

## Features
- 1 minute countdown
- Start button
- Auto stop when timer ends
- Dynamic timer updates using JavaScript
- Clean UI

## Technologies Used
- HTML
- CSS
- JavaScript

## What I Learned
- DOM Manipulation
- setInterval()
- Event Listeners
- Updating elements dynamically

## Project Goal
This project was created for practicing JavaScript and improving frontend development skills.
